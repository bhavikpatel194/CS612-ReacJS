-> MY FIRST REACT WEB PAGE:

1. The web page consists of header section, the heading describes what the succeeding information is all about.
2. It consists of main section, which has the main content of the page.
3. It includes side navigation section, which refrences to the more detailed description of the content stated in the main section.
4. It consists of a Footer section, that contains information such as document name, a graphic etc.


-> How to run ReactJS application:

1. Open the command prompt in your ReactJS application directory. (In this case, Open command prompt in the react-webpage directory.)
2. Type 'npm start' in the command prompt to run the ReactJS application.
